# System Defender (Cookie Game Extension)

이 프로젝트는 기존 Cookie Game을 확장하여 개발된 "System Defender" 웹 게임입니다.

## 🎮 주요 기능

-   **시스템 방어**: 몰려오는 바이러스와 버그(적)를 타자 및 수학 연산으로 처치하세요.
-   **다양한 모드**:
    -   ⌨️ **한글 타자 방어**: 떨어지는 단어를 타이핑하여 공격
    -   ✖️ **구구단 방어**: 구구단 문제를 풀어 공격
    -   ➕ **덧셈/뺄셈 방어**: 1~2자리수 덧셈/뺄셈 문제 해결 (5단계 난이도)
-   **성장 시스템**: 적을 처치하고 레벨업하여 다양한 등급(일반~전설)의 보상을 선택하세요.
-   **보스전**: 강력한 보스와의 전투, 특수 패턴 및 처치 실패 조건(Game Over) 적용.
-   **반응형 UI**: 태블릿 및 모바일 환경 지원 (분할 키보드).

## 🚀 실행 방법

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
